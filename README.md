# drex
DREX (Domestic Resource Estate eXchange) — Cohabitation and Non-Marital Property Agreement

A CC0 cohabitation contract template for unmarried partners who want clear property boundaries, non-marital intent, confidentiality carve-outs, and a defined separation framework.

## Scope
DREX is designed for **cohabitation** (living together) and is not a general “relationship agreement.”
It is not intended to govern child custody, child support, or other non-waivable rights.

## Usage
1. Copy `DREX_Cohabitation_Agreement.md` into Google Docs / Word.
2. Fill every placeholder like `[__________]` and `$[Amount]`.
3. Attach and initial exhibits (if used).
4. Have each party review with **independent counsel**.
5. Execute with signatures and (optionally) notarization.

## High-priority fields to fill
- Governing law + exclusive venue (Article XII)
- Arbitration locale (Article X)
- Residence address and definitions
- Support amount (if any)
- Relocation severance amount
- Cause thresholds ($ amounts)
- Reference Assets + baseline exhibits (if using Article V)

## Notes on lawful reporting
Do not remove the template’s Protected Disclosures / lawful reporting carve-outs (including sexual assault/harassment disclosure protections) without advice from counsel.
